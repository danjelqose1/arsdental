
# Ars Dental — Simple Website

This is a lightweight, fast, SEO‑friendly website for **Ars Dental**. It includes:
- Responsive layout
- EN/SQ/IT language toggle
- WhatsApp booking
- Embedded Google Map
- LocalBusiness (Dentist) JSON‑LD schema
- Clean HTML/CSS/JS (no frameworks)

## Quick Start
Open `index.html` in a browser.

## Customize
- Replace doctors' names, credentials and photos if available.
- Update opening hours in `index.html` and in the JSON‑LD schema.
- Set correct `email` and `website_url` in the HTML head.
- Replace social links in the schema `sameAs` array.

## Deploy (Netlify or GitHub Pages)
1. **Netlify**: Drag‑and‑drop this folder to Netlify. Done.
2. **GitHub Pages**:
   - Create a repo (e.g. `ars-dental-site`).
   - Push files.
   - Enable "Pages" in repo settings (deploy from `main` root).

## Domain & DNS
- Use or purchase the domain (e.g. `arsdental.al`).
- Point DNS to your hosting (Netlify/GitHub will show the records).

## SEO To‑Dos
- Write 3 blog posts targeting local keywords (optional later).
- Embed real Google reviews.
- Keep adding clinic photos monthly and post updates in Google Business Profile.
